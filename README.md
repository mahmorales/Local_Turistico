#  Local Turístico – Busan

##  Sobre o projeto
Este projeto é uma página web estática desenvolvida em **HTML e CSS**, que apresenta informações sobre a cidade de **Busan, Coreia do Sul**, destacando seus principais pontos turísticos e características culturais.  
O layout foi construído seguindo um guia de estilo no Figma da Rocketseat, com atenção especial a espaçamentos, tipografia e cores.

---

##  Estrutura visual
- **Container principal (`#page`)**: largura fixa de 1120px, centralizado na tela.  
- **Tipografia**: fonte *Open Sans*, importada do Google Fonts.  
- **Cores**:  
  - Títulos principais: `#E1624F`  
  - Textos: `#333`  
  - Fundo: `#F5F4F0`  
- **Espaçamentos principais**:  
  - Imagem → legenda: 24px  
  - Linha separadora: 40px em cima, 64px embaixo  
  - Título → parágrafo: 8px  
  - Parágrafo → imagem: 64px (primeiro destino) / 48px (demais destinos)  

---

##  Conteúdo
- **Imagem principal**: foto de Busan com legenda descritiva.  
- **Seções de destinos**: três blocos com imagem, título, descrição e lista de “Bom para”.  
- **Rodapé**: linha separadora, parágrafo final centralizado e assinatura “Feito com ❤️ pela Rocketseat”.

---

##  Estrutura de arquivos
- index.html   # Estrutura da página
- style.css    # Estilos e layout
- assets/      # Imagens e ícones (Busan, templos, parque, coração)

---

##  Como executar
1. Clone ou baixe este repositório.  
2. Abra o arquivo `index.html` em qualquer navegador moderno.  
3. Certifique-se de que a pasta `assets/` contém todas as imagens utilizadas.  

---

##  Aprendizados
- Uso de **margin e padding** para controlar espaçamentos verticais.  
- Aplicação de **classes específicas** para listas com cores diferentes.  
- Centralização de conteúdo com `margin: 0 auto`.  
- Estilização de bullets com `::marker`.  
- Organização semântica com `<section>`, `<figure>` e `<footer>`.
